# Schemas

This directory contains the core JSON schemas for validating data structures across repositories:

- `source.json`: Schema for source definitions in the research ledger.
- `claim.json`: Schema for claim definitions in the research ledger.
- `reference.json`: Schema for reference index entries and BibTeX metadata.

For detailed instructions on how validation works and how to add new schemas, please refer to the [Validation Guidelines](../docs/validation-guidelines.md).
